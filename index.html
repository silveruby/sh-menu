About

This is the demo for a jQuery Scroll Highlighted Menu plugin. How does it work? Well, every time you scroll to another section of the page, the plugin automatically updates the side menu to reflect that change.

It is pretty simple but I think it is a good way to interaction with users.

Quick note: there are some holes in this plugin. Because actually calculate the scroll base on half of the bowser height, if the first & last content div height is smaller than half of the browser height, then it might not work properly.

To use this, you'll need to include the latest jQuery file and the plugin - shmenu.jQuery.js - in your code. The plugin function is $(id).imenu().
